# Grupa3-Lambda
Ambasada - Vize

Aplikacija ce biti povezana sa web-om gdje ce se moci poslati 
zahtjev za vizu i ukoliko su ispunjeni uslovi radnik ambasade 
ce potvrditi zahtjev, pa se zahtjev ukljucuje u tombolu.
Za svaki korak procesa poslat ce se mail osobi koja je podnijela 
zahtjev da bi bila informisana. Vršit ce se i tombola za odabir osoba,
nakon kojeg ce se moci izdati vize najsretnijim prijavama. 
Postojat ce administratoski dio za dodavanje i brisanje racuna
uposlenih koji budu koristili ovaj app.
